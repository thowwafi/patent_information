# patent_information

## Installation Guide

1.a. Create virtualenv
    `python -m venv venv`
1.b. Activate virtualenv
    `venv\Scripts\activate` (Windows)
    `. venv/bin/activate` (Unix)
2. Upgrade pip
    `pip install --upgrade pip`
3. Install requirements
    `pip install -r requirements.txt`
4. Run script by year
    `python main.py 2015`
